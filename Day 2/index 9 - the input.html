        Thẻ label là thẻ nhãn thường dùng trong form
        Thẻ input có type="" đây là 1 ô input để nhập liệu thông thường
        các thuộc tính cảu thẻ input có type="text"
        1-thẻ input luôn có thuộc tính type để chỉ ra kiểu của ô input
        2-thuộc tính name="" : tên cảu ô input
        3-thuộc tính value="" : chứa giá trị của ô input
        4-thuộc tính placeholder="" : gợi ý cho ng dùng gõ gì vào khi giá trị value ccar ô input rỗng
